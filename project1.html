Goal

Research a topic from a provided list and gamify the topic around the core idea

Description

This project is the result of an assignment given to us in our gamification class by Sheridan. We were tasked to research a topic from a provided list and construct a research paper with an accompanying idea for a game with the results of the research at its core. After all pitches were given, we were given control over who we wanted on our team and which pitch to use.

Ultimately, my team decided to go with the pitch that talked about the current state of housing in Ontario and the struggles with keeping a home amidst the rising housing costs. I am the programmer and UI designer on the team. I am responsible for the UI, mechanic implementation, debugging, scripting, source control management and more.

Since the project is a visual novel, the team agreed to implement and learn Fungus, a visual storytelling and scripting tool that comes with a dialogue system so I can focus other features and polish of other mechanics.

Accomplishments

  • Implemented Fungus into our project
  • Implemented custom code using Fungus's Command system
  • 950+ lines of scripting
  • Hover mechanic that hints what each choice does to resources
  • Multiple UI iterations based on feedback from play tests
  • Ability to implement and use new APIs
  • Ability to script in C#
  • Scene management and game object scripting
  • Implement several systems that work together
  • Understanding of version control usage and its practices
