Goal

Create a dynamic, interactive menu system that allows the users to customize different weapons.

Description

Gunsmith is a project created for an assignment in an asset creation class at Sheridan College. The assignment last the course of the semester where we modeled a gun and attachment, textured them, and imported the assets into Unreal Engine to create a menu that allowed the user to customize the selected weapon, paint, and attachment. Collaboration was allowed in the form of sharing assets for more customization options.

During the course of this project, I modeled and texture two weapon assets (a gun and attachment) and imported them into Unreal. I took advantage of the Unreal Marketplace for additional assets for the scene. Sound and music were taken online. The customization system, HUD, and all accompanying code were created by me. The other weapons and attachments were created by other students (see Collaborators below).

Accomplishments

  • Knowledge in navigation the Unreal Editor interface
  • Understanding of Unreal's Blueprint system
  • 3D asset creation (Maya)
  • Understanding of asset creation concepts (textures, materials, normal maps, etc)
  • Material handling and manipulation
  • Implemented multiple assets such as meshes, textures, and sound files
  • Camera zoom to better show what you are doing
  • Dynamic menu that reacts to the gun being modified
  • Create a particle system and camera bob to add movement to the screen
  • Added polish to the demo with sound such as button clicks, ambience, and music
