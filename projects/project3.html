Goal

Design a level that is inspired by game.

Description

This project was a result of an assignment given for a level design class at Sheridan College. The assignment last the course of the semester and started from an initial case study of level in a game. We were given control the tools we would use to design the level.

My case study was on Metroid Prime (2002) and its Chozo Ruins level. I use the level as a reference to create a small metroidvania inspired demo. I also decided to lean more into the programming side and created abilities that the player can pick up that will open up the level to them.

This project helped me familiarize myself with Unreal and its Blueprint System. I plan to use this knowledge to help my progress in learning C++.

Accomplishments

  • Created multiple systems using Blueprints
  • Altered existing systems to achieve my design goals
  • Created enemies that will attack the player once spotted
  • Created custom 3D assets with materials
  • Basics of level management
  • Ability to use and navigate the Unreal Editor
  • Fundamental understanding of the Blueprint system
  • Basic of AI implementation including path-finding, behaviour trees, and AI controllers
  • Knowledgeable in Maya and Blender to create assets
  • Understanding of asset creation concepts (textures, materials, normal maps, etc)
