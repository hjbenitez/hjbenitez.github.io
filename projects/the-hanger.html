COLLEGE PROJECT

This project was a result of an assignment given for a level design class at Sheridan College. This assignment drew inspiration from Metroid Prime's Chozo Ruins. I use the level as a reference to create a small metroidvania inspired demo. I also decided to lean more into the programming side and created abilities that the player can pick up that will open up the level to them.
